# Minecraft_Unarchived
LDTで作られた、呪いのMinecraftバージョン第4弾 実況は現時点で不可です
Minecraft 1.2.5 Unarchived

---

## これは何？
- Java 8で動作する **MultiMC** または **PrismLauncher** 用のmod形式の呪いのMinecraftです。  
- プレイするには **Minecraftアカウント（Microsoft）** が必要です（有料）。すでにアカウントを持っている方は、そのアカウントでプレイ可能です。  
- **BE（スマホ、Switch）のアドオンではありません**。ご注意ください。

## 動かし方

### クライアント
**必要なもの:** Java 8（Zulu推奨）  
[Zulu Java 8 ダウンロード](https://www.azul.com/downloads/?version=java-8-lts&os=windows&architecture=x86-64-bit&package=jdk#zulu)  

1. クライアントzipファイルをダウンロード：  
[Unarchived.zip](https://github.com/LDNDevelopTeam/Minecraft_Unarchived/releases/latest)

2. **MultiMC** または **PrismLauncher** をインストール：  
- [MultiMC](https://multimc.org/#Download)  
- [PrismLauncher](https://prismlauncher.org)  

   インストール後、Minecraftアカウントでログインし、ダウンロードしたzipファイルを **ドラッグ＆ドロップ** します。  

3. インストール完了後、Javaの設定で先ほどインストールしたZulu Java 8を指定。メモリ割り当てを必要に応じて変更し、起動ボタンをクリック。  

4. プレイ開始！
